# Code Of Conduct

## Our standard

This project expects respectful, technically honest collaboration.

Participants should:

- critique ideas, code, and decisions without attacking people
- assume mistakes are fixable and discuss them directly
- keep disagreements grounded in evidence and reproducible behavior
- avoid harassment, discrimination, intimidation, and personal abuse

## Scope

This applies to repository discussions, issues, pull requests, releases, and project-adjacent community spaces operated for PortUI.

## Enforcement

Maintainers may remove comments, close threads, or block participation when behavior harms the project or its contributors.

## Reporting

If a problem cannot be handled safely in public, contact the maintainer through the repository owner profile on GitHub and include enough context to investigate.
